# Weather-API
A Django REST Framework-based Weather API with Token Authentication, request logging, and a simple frontend interface. Users can log in to receive a token, access protected endpoints, view a list of cities, and retrieve weather information. All requests are logged with user, timestamp, IP, endpoint, and method.
✔ Token-based Authentication (DRF)

✔ Protected REST API Endpoints

✔ City List API (/cities/api/)

✔ Weather Info API (/weather/api/)

✔ Case-insensitive city search

✔ Latest weather record retrieval

✔ Centralized request logging system

✔ Admin panel log management

✔ Simple frontend integration (HTML + JS)
